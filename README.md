## Using MySQL in Django-Python

#### Install dependencies

``` bash

# install virtualenv
	$ cd api/
	$ python3 -m venv ../VENV

# install package
	$ source ../VENV/bin/activate
    $ pip install -i https://pypi.tuna.tsinghua.edu.cn/simple  -r requirement.txt 

```

### Run

#### From terminal

	$ source ../VENV/bin/activate
	$ python manage.py runserver 0.0.0.0:8000
